## USAGE INSTRUCTIONS

1. **For Lab Technicians:**
   - Copy the relevant section based on the test ordered
   - Fill in the blank spaces with actual test results
   - Ensure units match the specified format
   - Compare results with normal ranges provided

2. **For Doctors:**
   - Use these templates when ordering comprehensive panels
   - Reference normal ranges for interpretation
   - All values are provided with standard international units

3. **For System Integration:**
   - These templates can be stored in the database as text
   - Use HTML formatting for better presentation
   - Link each template to appropriate lab category
   - Allow customization per hospital requirements

## NOTES

- Normal ranges may vary slightly between laboratories
- Reference ranges provided are for adult patients
- Pediatric and geriatric ranges may differ
- Always consult your laboratory's specific reference ranges
- Critical values should be reported immediately to the physician
- All tests should be performed according to standard operating procedures
