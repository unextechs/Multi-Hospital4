Easy Setup for Single or Multi-Hospital Use

For Single Hospital:

  1. Install the application.
  2. Login as admin using the provided credentials.
  3. Update your details in the profile module.
  4. Start managing your hospital efficiently!

For Multiple Hospitals:

  1. Install the application.
  2. Login as super admin and create hospitals from the dashboard.
  3. Each hospital gets its own software with unique login credentials.
  4. Manage multiple hospitals effortlessly through one platform!
